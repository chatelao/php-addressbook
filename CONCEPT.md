# Concept - PHP Address Book

## Business Cases

1.  **Centralized Contact Management**: Providing a single, web-accessible source of truth for personal and professional contact information.
2.  **Cross-Device Synchronization**: Enabling seamless contact synchronization across smartphones and other mobile devices using the ActiveSync (Z-Push) protocol.
3.  **Social Media & Third-Party Integration**: Streamlining user authentication and data enrichment through integration with major social platforms like Google, Facebook, and Microsoft.

## Use Cases

### 1. Contact Lifecycle Management (CRUD)
- **Primary Actor**: User
- **Goal**: Maintain accurate contact details.
- **Flow**:
    - User adds a new contact with multiple phone numbers, emails, and a photo.
    - User searches for a contact and updates their address.
    - User deletes obsolete contact records.

### 2. Group Organization
- **Primary Actor**: User
- **Goal**: Categorize contacts for easier management and bulk communication.
- **Flow**:
    - User creates a "Work" group and assigns relevant contacts to it.
    - User filters the main list to show only the "Family" group.
    - User exports all contacts in a specific group to a CSV file.

### 3. Birthday & Anniversary Tracking
- **Primary Actor**: User
- **Goal**: Never miss a contact's birthday.
- **Flow**:
    - User views the upcoming birthdays list.
    - User subscribes to the iCalendar feed generated by the application.

### 4. Data Interoperability (Import/Export)
- **Primary Actor**: User / External System
- **Goal**: Move contact data in and out of the application.
- **Flow**:
    - User imports contacts from an old CSV or LDIF file.
    - User downloads a vCard for a contact to import into their local mail client.

## High-Level Architecture

The application follows a classic LAMP stack pattern with integrated specialized libraries:

- **Frontend**: Server-side PHP generating HTML/CSS, enhanced with jQuery 1.8.2 and DataTables 1.9.4 for interactivity.
- **Backend**: Core PHP logic (compatible with PHP 5.4+).
- **Database**: MySQL schema for persistent storage.
- **Integrations**:
    - **Z-Push**: Provides the ActiveSync gateway.
    - **HybridAuth**: Manages social authentication flows.
    - **Libraries**: parseCSV and PHP Excel Reader for file processing.
